
# Hermes Companion

**Self-installing GUI + Daemon for Hermes Agent**

One-click installer that turns any Mac/Windows/Linux machine into a secure node in your personal Hermes network.

- Clean, minimal chat UI (Tauri + Svelte)
- Background daemon that registers the machine with your central Hermes on DigitalOcean
- Central Hermes gains full SSH + tool access to the machine
- Works from anywhere — email the installer, run once, done

## Quick Start (for users)
1. Download the installer for your OS
2. Run it
3. Enter your central Hermes server details once
4. Done — the machine is now part of your network
